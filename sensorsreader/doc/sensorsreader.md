# datax-sensors-plugins

eg:

```json
{
    "name": "sensorsreader",
    "parameter": {
        "username": "",
        "password": "",
        "column": [],
        "connection": [
            {
                "jdbcUrl": [],
                "table": []
            }
        ],
        "where": ""
    }
}
```
